# Portfolio - Vo Truong

Un portfolio moderne et performant construit avec Next.js 14+, TypeScript, Tailwind CSS, GSAP et Lenis.

## 🚀 Technologies

- **Next.js 14+** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **GSAP** (avec ScrollTrigger)
- **Lenis** (Smooth Scroll)
- **Lucide React** (Icônes)

## 📦 Installation

```bash
npm install
```

## 🏃 Développement

```bash
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## 🎨 Fonctionnalités

- ✅ Header avec menu burger animé pour mobile
- ✅ Hero Section avec animations stagger
- ✅ Marquee infinie avec GSAP
- ✅ Section About avec effet de scroll
- ✅ Skills avec effet de blur au hover
- ✅ Projects avec image flottante au survol
- ✅ Services avec cartes animées
- ✅ Footer avec bouton magnétique
- ✅ Smooth scroll avec Lenis
- ✅ Design responsive
## 📱 Responsive

Le site est entièrement responsive avec des adaptations spécifiques pour mobile, tablette et desktop.
