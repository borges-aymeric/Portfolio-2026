# Fonts Directory

Placez le fichier de police Sporteria ici :

- `Sporteria-Regular.ttf`

Vous pouvez télécharger Sporteria depuis :
- https://exfont.com/sporteria-regular.font (usage personnel)
- Ou depuis d'autres sources fiables

Une fois le fichier téléchargé, renommez-le en `Sporteria-Regular.ttf` et placez-le dans ce dossier.
